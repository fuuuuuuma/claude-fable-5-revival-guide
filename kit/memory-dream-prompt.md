# 🎁 設定修復プロンプト（memory-dream）

Claude Code の CLAUDE.md / Skills / メモリに溜まったノイズ（期限切れの相対日付・削除済み手順への参照・重複・矛盾）を一括整理するプロンプトです。

**受け取り方**: 下のコードブロック右上のコピーボタンでコピーするか、このファイルをそのままダウンロードしてください。

**使い方**: Claude Code にそのまま貼り付けるだけ。繰り返し使うなら `.claude/skills/memory-dream/SKILL.md` として保存しておくと、「memory-dream を実行して」の一言で呼び出せます。

```text
memory-dream を実行して、現在の記憶階層全体（CLAUDE.md / Skills / メモリファイル）を整理してください。

手順:
1. Mine: 直近のセッション記録から、繰り返し出た指摘・確定方針・新事実だけを抽出（1回限りのメモは無視）
2. Consolidate: 既存ファイルへマージ。相対日付→絶対日付、矛盾は最新で上書き、死んだ参照は削除
3. Dedup & Resolve: 上位ファイルにあるルールを下位から削除。メタ説明は残さない
4. Prune & Index: 索引は200行以内に。ノート一覧を実ファイルから再生成

鉄則: 経緯・失敗談・履歴は書かない。現時点で正しい知見・ルール・再現手順のみ残す。
出力はレビューできる形で、変更をコミットとして提示してください。
```

**ポイント**: 出力は自分でレビューしてから確定してください（勝手に確定させない）。モデルを Fable 5 など上位モデルに切り替えたタイミングで実行すると、広い文脈での矛盾検出が期待できます。
